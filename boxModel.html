this box model is how box can be wrap around every HMTL element. It consists of borders, padding, margins, and the actual content

  • Content: This is where the text and image appear
  • Padding: this clears an area around the content. the padding is transparent
  • Border: this goes around the padding and content
  • Margin: this clear an area outside the border, The margin is also transparent
    • The box model allows us to add a border around elements, and to define space between elements
