(BOM) Browser Object Model while (DOM) is Document Object Model

whenever a web page is loaded, the browser creates a DocumentObject Model of the page

    List of Document Elements
  • Root element: ( < HTML > )
  • Root element has 2 children: ( < head > and < body > )
  • Head element has 3 or more children: ( < title > and < Text i.e links >, < meta i.e the logical elements > )
  • Body element has attribute and text

With this Object Model Javascript creates a dynamic HTML

    Things javascript can do:
  1. it can change all HTML element in the page
  2. it can change all HTML attributes in the page
  3. it can change all the Css style in the page
  4. it can remove existing HMTL elemnts and attrribute in the page
  5. it can also add new HMTL elements and attributes in the page
  6. it can react to all existing Html events in the page i.e call to action eg: button onClick
  7. it can create new HTML events on the page

DOM Interface

  • The HTML element as Objects
  • The properties of all HTML elements
  • The moethods to access all HTML elements
  • The events of all HTML elements

BOM Categories

  1. Js window: window.open() which is used to open a new window, window.close(), window.moveTo(), window.resizeTo()
  2. js screen: screen.width, screen.height, availWidth, availHeight, colorDepth (24bits = True color, 32bits(Deep color)), pixelDepth

  3. Locattion(URL): has href(current URL page), hostname (domain name of web host), pathname(current page file path and filename), protocol(returns the web protocol used(http: or https) which also be the ip address, assign() loads a new document

  4. History: contains the browser History it has history.back() which means same as clicking back in the browser and same as forward()
  5. Navigator: this object contains information about the visitor's browser, properties: Navigator.appName, navigator.appCodeName, navigator.platform

  6. Js Popup Alert
